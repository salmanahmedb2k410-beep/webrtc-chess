## Introduction
This project allows users to play chess online using WebRTC technology. It provides a real-time multiplayer experience without needing to reload the page.

## Features
- Real-time gameplay
- User authentication
- Spectator mode

## Installation
To set up the project locally, follow these steps:
1. Clone the repo:
   ```bash
   git clone https://github.com/nightrider99/webrtc-chess.git
   ```
2. Navigate to the project directory:
   ```bash
   cd webrtc-chess
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
Visit `http://localhost:3000` in your web browser to play.
