📡 LAN Play Guide

What is LAN Play?
Two devices on the same WiFi or hotspot network can play each other. No internet is needed after setup.
Host (White):
① LAN Multiplayer → Host Game
② Click Start Hosting → copy code
③ Send code via WhatsApp/SMS to friend
④ Paste their reply → Connect → Game starts!
Guest (Black):
① LAN Multiplayer → Join Game
② Paste host code → Connect
③ Copy your answer code → send to host
④ Wait for host to accept → Game starts!
